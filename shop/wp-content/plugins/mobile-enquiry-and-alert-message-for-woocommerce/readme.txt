Mobile Enquiry and Alert Message for Woocommerce is used to get a enquriy from user directly to your whatsapp for product, cart and order detail etc!

== Description ==
This plugin provides you options to get enquiries from users via whatsapp. There are lots of ways to get inquiry from users. Mobile Enquiry and Alert Message for Woocommerce provides you really popular What platform for enquiry, that is Whatsapp. You can get an enquiry message using Whatsapp for all your woocommerce products. Set enquiry button color, text, size of it dynamically and also can set different Whatsapp enquiry buttons for different pages.

You will have the flexibility to display the Whatsapp enquiry button for a particular product and category. Hide the add-to-cart and checkout button on product and cart pages.

This plugin is very easy to use. Enter your Whatsapp number and select the position where to show your Whatsapp enquiry button and here you go!

Functionality for setting user role wise Whatsapp enquiry button. Can select multiple user roles or simply select non-logged-in users.

Set a Whatsapp enquiry button in your main Woocommerce pages like Single Product, Cart, Checkout, Order, My Account pages with dynamically set button text, color, and size. Option to hide add to cart and checkout button.

You can select your favorite button position and also can set a dynamic message body for each page.

<h4>You can use shortcodes to display whatsapp button</h4>
1. Single Product: <code>[mmwea_single_product_wh_btn product_id="123"]</code>
1. Cart: <code>[mmwea_cart_wh_btn]</code>
1. Checkout: <code>[mmwea_checkout_wh_btn]</code>
1. Orders: <code>[mmwea_order_wh_btn order_id="232"]</code>
1. My Account Orders: <code>[mmwea_my_account_wh_btn order_id="229"]</code>
This Shortcode displays whatsapp button for diffrent pages. Like Single Product, Cart, Checkout, Orders.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

After Plugin Active go to Woocommerce-> Mobile Enquiry and Alert Message for Woocommerce
